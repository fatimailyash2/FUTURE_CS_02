# FUTURE_CS_02

This task presents a phishing detection and awareness analysis performed on a collection of email samples obtained from the phishing_pot database. The emails were examined using publicly available tools, including MxToolbox, alongside manual inspection techniques to identify potential phishing indicators.

This report contains a detailed analysis of five phishing email samples, focusing on key elements such as email headers, sender and return-path inconsistencies, authentication mechanisms (SPF, DKIM, and DMARC), relay paths, and message content. Each email is evaluated for suspicious patterns, social engineering techniques, and technical anomalies that indicate malicious intent.

This report also highlights common phishing strategies, including spoofed domains, lack of proper authentication, misleading subject lines, urgency-based tactics, and deceptive links. Additionally, it provides prevention and awareness guidelines aimed at helping users recognize and respond to phishing attempts effectively.
